# Student Course Management System

A MERN stack application with authentication and course management.

## Features

- Student register and login
- Create course
- View courses
- Delete course
- Search courses

## Tech Stack

- Frontend: React (CRA)
- Backend: Node.js, Express
- Database: MongoDB (Mongoose)
- Auth: JWT

## Run Locally

### 1) Backend

```bash
cd backend
npm install
npm run dev
```

### 2) Frontend

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on `http://localhost:5173` and backend on `http://localhost:5000`.
